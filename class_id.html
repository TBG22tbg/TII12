Trabalhando com ID e classe

Seletores ID

É usado um seletor de id em um elemento HTML parad deixar ele único e especifico.

Deve ser usado uma única vez na página.

Exemplo

USANDO UM ID

Seletores de classes

O seletor de classe seleciona elemento com um atributo de classe especifico.

Pode ser usuado mais de uma vez na página.

Exemplo:

USANDO CLASSE

USANDO CLASSE

USANDO CLASSE

OUTRO TEXTO
